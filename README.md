# Fissler - Shopify Theme

## Prerequesites

**Node.js** Version 20 or higher

**yarn**
```
brew install yarn
```

**Git**

**Shopify CLI**
```
brew tap shopify/shopify
brew install shopify-cli
```

## Local Development

1. Clone the repository:

```
git clone git@github.com:Latori-GmbH/fissler-shop.myshopify.com.git
```

2. Start development process

```
yarn theme:eu:dev
```

## Development Guidelines

- Follow Shopify's [Liquid Best Practices](https://shopify.dev/docs/storefronts/themes/best-practices)
- Write JavaScript comments using [JSDoc](https://jsdoc.app/)

## Tooling and Useful Plugins
